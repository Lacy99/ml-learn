Learning Set of Rules

การเรียนรู้เซตของกฎและกฎความสัมพันธ์เชิงข้อมูล

1 การเรียนรู้เซตของกฎ (Learning Set of Rules)

แทนที่จะสร้างรูปแบบโครงสร้างต้นไม้เหมือน Decision Tree อัลกอริทึมกลุ่มนี้จะเลือกดึงข้อมูลออกมาสร้างเป็นกลุ่มของกฎกติกาในรูปแบบ **"ถ้า... แล้ว..." (IF - THEN Rules)** เพื่อให้เกิดความง่ายต่อมนุษย์ในการอ่านทำความเข้าใจและนำไปวิเคราะห์ต่อ

IF วันหยุด = ใช่ AND สภาพอากาศ = ปลอดโปร่ง THEN กิจกรรม = ท่องเที่ยว

อัลกอริทึม Sequential Covering

เป็นวิธีการหลักในการสร้างกฎ โดยการกำหนดชุดกฎหนึ่งข้อที่จำแนกเป้าหมายได้ถูกต้องที่สุด จากนั้นจะทำการ **"คัดกรองตัวอย่างข้อมูลที่ตรงกับกฎข้อนั้นออกไป" (Covered Cases)** แล้วทำการวนซ้ำสร้างกฎข้อถัดไปเรื่อยๆ จนกระทั่งไม่มีข้อมูลตัวอย่างเหลือในคลังเก็บ

อัลกอริทึมสร้างกฎที่ได้รับความนิยมคือ **CN2 Algorithm** ซึ่งออกแบบมาแก้ปัญหาเสียงรบกวนในข้อมูลโดยใช้ค่า Entropy หรือหลักความน่าจะเป็นเข้ามาวิเคราะห์เพื่อให้กฎที่สกัดออกมามีความเสถียรยิ่งขึ้น

2 กฎความสัมพันธ์และอัลกอริทึม Apriori (Association Rules)

กฎความสัมพันธ์ (Association Rules) เป็นการวิเคราะห์หาความเชื่อมโยงของสิ่งของต่าง ๆ ที่มักจะเกิดขึ้นพร้อมกัน (เช่น การหาความเกี่ยวโยงของสินค้าที่ลูกค้าเลือกซื้อพร้อมกันในตะกร้าซูเปอร์มาร์เก็ต หรือ Market Basket Analysis)

เมตริกสำคัญ 3 ประการในการวัดคุณภาพของกฎ

พิจารณากฎความสัมพันธ์ในรูปแบบ $X \rightarrow Y$ ($X$ ส่งผลให้ซื้อ $Y$):

1. Support (ค่าสนับสนุน)

สัดส่วนของธุรกรรมทั้งหมดที่มีทั้ง $X$ และ $Y$ อยู่ร่วมกัน

$$Support(X \rightarrow Y) = \frac{Transactions(X \cup Y)}{Total \, Transactions}$$

2. Confidence (ค่าความเชื่อมั่น)

โอกาสที่จะซื้อ $Y$ เมื่อมีการซื้อ $X$ เกิดขึ้นแล้ว

$$Confidence(X \rightarrow Y) = \frac{Support(X \cup Y)}{Support(X)}$$

3. Lift (ค่าแรงจูงใจ)

ตัวชี้วัดความสัมพันธ์ว่าการเกิด $X$ ส่งเสริมให้เกิด $Y$ มากน้อยเพียงใด (ค่า Lift > 1 หมายถึงเชิงบวก, Lift = 1 เป็นอิสระต่อกัน)

$$Lift(X \rightarrow Y) = \frac{Confidence(X \rightarrow Y)}{Support(Y)}$$
